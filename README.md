## ðŸš€ How to Run Locally
If you want to run this code on your own machine:

1.  **Clone the repo**
    ```bash
    git clone [https://github.com/SoulScannerOfficial/soul-scanner.git](https://github.com/SoulScannerOfficial/soul-scanner.git)
    ```

    ```bash

    ```

    Create a `.env` file and add your Google Gemini API key:
    ```bash
    SOULSCANNER=your_google_api_key_here
    ```
4.  **Run the server**
    ```bash
    npm start
    # or use Vercel CLI
    vercel dev
    ```
